4.0表格

十四、表格

表头+表分割线+表数据 即可生成一个表
第二行 表分割线 分割表头和表数据
- 语法：

表一：
| col 1 | col 2 | col 3 |
| --- | --- | --- |
| cell1 | cell2 | cell3 |
| cell4 | cell5 | cell6 |

有一个-就行，为了 对齐，多加了几个，注意中间必须连贯不能有空格。
文字默认居右对齐
- 两边加: 表示文字居中
- 右边加: 表示文字居右
- 左边加: 表示文字居左

表二：  
|字段|类型|空|默认|注释|
|:----    |:-------    |:--- |---|------      |
|uid    |int(10)     |否 |  |             |
|username |varchar(20) |否 |    |   用户名  |
|password |varchar(50) |否   |    |   密码    |
|name     |varchar(15) |是   |    |    昵称     |
|reg_time |int(11)     |否   | 0  |   注册时间  |
